### 핵심 알고리즘
- **상단 삼각형** (1~5줄): `i`번째 줄에 전각 공백 `　` `(size - i)`개 + ★ `(2 * i - 1)`개 출력
- **하단 역삼각형** (4~1줄): 동일한 공식을 `i` 감소 방향으로 반복
- 총 9줄 출력, 최대 너비 9개 ★

### 코드 구조
```
DiamondStar.java
├── size = 5 (다이아몬드 크기)
├── 상단 삼각형 루프 (i: 1 → 5)
│   ├── 전각 공백(　) 출력 루프
│   └── ★ 출력 루프
└── 하단 역삼각형 루프 (i: 4 → 1)
    ├── 전각 공백(　) 출력 루프
    └── ★ 출력 루프
```

## 학습 포인트

1. **중첩 for문**: 외부 루프(줄) + 내부 루프(공백/별) 조합
2. **수학적 패턴**: `(2 * i - 1)` 공식으로 홀수 개수 생성
3. **증감 방향**: 상단은 증가(`i++`), 하단은 감소(`i--`)로 대칭 구현
4. **문자 너비**: 전각 문자(★) 사용 시 공백도 전각(　)으로 맞춰야 대칭이 유지됨
