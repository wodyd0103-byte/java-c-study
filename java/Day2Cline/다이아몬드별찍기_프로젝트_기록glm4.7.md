## 💻 구현 코드

### 파일: `src/DiamondStar.java`

```java
void main() {
    int size = 5; // 다이아몬드 크기 (위쪽 삼각형의 행 수)
    
    // 위쪽 삼각형 (1행 ~ size행)
    for (int i = 1; i <= size; i++) {
        // 선행 공백 출력
        for (int j = 1; j <= size - i; j++) {
            IO.print(" ");
        }
        
        // 별 출력
        for (int k = 1; k <= 2 * i - 1; k++) {
            IO.print("★");
        }
        
        IO.println();
    }
    
    // 아래쪽 삼각형 (size+1행 ~ 2*size-1행)
    for (int i = size + 1; i <= 2 * size - 1; i++) {
        // 선행 공백 출력
        for (int j = 1; j <= i - size; j++) {
            IO.print(" ");
        }
        
        // 별 출력
        for (int k = 1; k <= 2 * (2 * size - i) - 1; k++) {
            IO.print("★");
        }
        
        IO.println();
    }
}
```

## 🔧 기술적 분석

### 알고리즘

#### 위쪽 삼각형 (행 1-5)
- **선행 공백:** `size - 행번호` 만큼
- **별 개수:** `2 × 행번호 - 1` 개

| 행 | 공백 | 별 개수 | 출력 |
|----|------|---------|------|
| 1 | 4 | 1 | `    ★` |
| 2 | 3 | 3 | `   ★★★` |
| 3 | 2 | 5 | `  ★★★★★` |
| 4 | 1 | 7 | ` ★★★★★★★` |
| 5 | 0 | 9 | `★★★★★★★★★` |

#### 아래쪽 삼각형 (행 6-9)
- **선행 공백:** `행번호 - size` 만큼
- **별 개수:** `2 × (2 × size - 행번호) - 1` 개

| 행 | 공백 | 별 개수 | 출력 |
|----|------|---------|------|
| 6 | 1 | 7 | ` ★★★★★★★` |
| 7 | 2 | 5 | `  ★★★★★` |
| 8 | 3 | 3 | `   ★★★` |
| 9 | 4 | 1 | `    ★` |

### 코드 구조

1. **변수 선언**
   - `size = 5`: 다이아몬드 크기 설정

2. **위쪽 삼각형 루프**
   - 외부 for문: 행 반복 (1부터 size까지)
   - 내부 for문 1: 선행 공백 출력
   - 내부 for문 2: 별 출력

3. **아래쪽 삼각형 루프**
   - 외부 for문: 행 반복 (size+1부터 2*size-1까지)
   - 내부 for문 1: 선행 공백 출력
   - 내부 for문 2: 별 출력

### 주요 로직

// 아래쪽 삼각형 별 개수 계산
