# 자바 · C 학습 기록

자바 심화 과정과 C 언어 기초 실습 코드 모음입니다.

## 구성

- **java/** — 자바 심화 과정 소스 (변수·연산자 → 반복문·배열 → 클래스·상속 → 파일 I/O → OpenAPI·JSON 파싱, 실습: 구구단·달력·영수증·성적집계)
- **javac/** — 컴파일 실습
- **c/** — C 포인터 기초 실습 (주소·역참조·배열-포인터 관계·포인터 연산)
- **html-tagtable-frame/**, **html-tagtable-css/** — 자바 과정 HTML/CSS 파트 실습 (태그 사전 · 프레임 · 스타일)

> 빌드 산출물(.class, .exe)은 제외되어 있습니다.
